# Лицензия ЛЕВИ (Личная Экспериментальная Вольная Инициатива)

**Версия 1.0, декабрь 2025 года**

## Преамбула

Настоящий программный продукт представляет собой набор инструментов для автоматизации создания учебных отчетов, включающий линт-средства для проверки документов в формате LaTeX, средства автоматической генерации графиков и систему сборки, использующую сторонние компиляторы. Проект разработан исключительно в личных целях автора репозитория и не претендует на статус профессионального программного обеспечения.
